HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar e estruturar páginas da web. É uma linguagem fundamental para o desenvolvimento web e é interpretada pelos navegadores para exibir conteúdo na forma de texto, imagens, vídeos e outros elementos interativos.

Estrutua de um documento HTML

    <!DOCTYPE html>
    <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-
            width, initial-scale=1.0">
            <title>Document</title>
            <link rel="stylesheet" href="style.css">
            <!-- Link de CSS -->
        </head>
        <body>
            <h1>Exemplo de título</h1>
            <p>Exemplo de parágrafo</p>
            <!-- Exemplo de comentário -->
        </body>
    </html>

A principal característica da atual versão do HTML é o uso de tags com a intenção de atribuir significados, ou seja, o uso de semântica. Nas versões mais antigas a estrutrua de um documento HTML era feito sem atribuição de significado, da seguinte forma:

    <html>
        <head>
            <title>Site com HTML4</title>
        </head>
        <body bgcolor="blue">
            Eu moro na <u>Rua dos Capixabas, 229 -
            Botafogo - RJ</u>
            <marquee behavior="" direction="right">
                Teste de Marquee
            </marquee>
        </body>
    </html>
CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência e o layout de elementos em uma página da web. É uma das tecnologias principais para estilização e design web e trabalha em conjunto com o HTML para criar páginas web visualmente atraentes e responsivas.

Aqui estão alguns pontos-chave sobre o CSS:

  1. Seletores: O CSS usa seletores para escolher elementos HTML aos quais se aplicarão as regras de estilo. Os seletores podem ser baseados em elementos HTML, classes, IDs, atributos e outras características. Por exemplo, um seletor de classe pode ser definido como ".classe" e um seletor de ID pode ser definido como "#id".
  2. Propriedades e valores: As propriedades CSS são usadas para definir o estilo de um elemento selecionado. Cada propriedade tem um valor associado que determina como o elemento será estilizado. Por exemplo, a propriedade "color" define a cor do texto, a propriedade "font-size" define o tamanho da fonte e assim por diante.
  3. Regras CSS: As regras CSS consistem em um seletor, propriedades e valores. Elas são usadas para aplicar estilos a elementos HTML específicos.

Estrutua das declarações em CSS

    /* Exemplo de comentário */
    h1 {
        font-size: 2em;
        color: blue;
    }

    p {
        color: white;
        background-color: black;
    }

Outra característica do CSS é a responsividade, o CSS permite criar layouts responsivos, adaptáveis a diferentes tamanhos de tela e dispositivos. Com o uso de regras de mídia, é possível aplicar estilos específicos para diferentes resoluções e condições de visualização, oferecendo uma experiência adequada em dispositivos móveis, tablets e desktops.

JavaScript

JavaScript é uma linguagem de programação amplamente usada para desenvolvimento web. Ela permite a criação de conteúdo interativo e dinâmico em páginas da web, oferecendo recursos de programação para manipular elementos HTML, controlar eventos, processar dados e interagir com servidores.

Estrutua de um código em JavaScript

    // Exemplo de comentário
    let x = 1;
    let y = 3;
    function somar(a=0, b=0) {
        return a + b;
    }
    console.log(soma(x, y)); // saída: 4

O JavaScript também permite que você responda a eventos do usuário, como cliques de mouse, pressionamentos de teclas e movimentos do mouse. Você pode atribuir funções JavaScript a esses eventos para criar respostas interativas, como alterar o conteúdo exibido, exibir mensagens de alerta, enviar formulários e muito mais.

Node.js

Node.js é um ambiente de tempo de execução (runtime) de JavaScript baseado no motor V8 do Google Chrome. Ele permite que você execute código JavaScript fora do navegador, diretamente no servidor. Com o Node.js, você pode criar aplicativos de servidor, scripts de linha de comando e até mesmo aplicativos de desktop.

O Node.js é amplamente adotado pela comunidade de desenvolvedores e tem um ecossistema robusto de pacotes e bibliotecas disponíveis. Ele é usado para criar uma variedade de aplicativos, desde servidores web simples até aplicativos em tempo real complexos. Para começar a usar o Node.js, você precisará instalá-lo em seu computador e utilizar um editor de texto ou uma IDE para escrever seu código JavaScript.

React

React é uma biblioteca JavaScript de código aberto usada para criar interfaces de usuário (UI) interativas e reutilizáveis. Desenvolvida pelo Facebook, o React permite construir aplicativos web modernos e escaláveis, com uma abordagem baseada em componentes.

O React utiliza um Virtual DOM (DOM Virtual) para atualizar eficientemente a interface do usuário. O Virtual DOM é uma representação leve da estrutura do DOM atual, que é comparada com a versão anterior para determinar as alterações necessárias. Isso resulta em atualizações mais eficientes e melhora o desempenho do aplicativo.
